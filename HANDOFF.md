## Data Source

The dashboard reads AI news from:
- **Location**: `/workspace/memory/ai-news-YYYY-MM-DD.md`
- **Format**: Markdown files generated by AI News Daily cron job
- **Update Schedule**: Daily at 21:00 SGT
- **Retention**: Latest 5 files are displayed

## Known Limitations

1. **Data Availability**: No news displayed until first AI News Daily cron job runs
2. **Server-Side Only**: Dashboard doesn't support client-side updates (requires page refresh)
3. **Single Page**: Only one page (home) is implemented
4. **Test Coverage**: Basic smoke test only (needs expansion)
